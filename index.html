Join the Fun of Secret Santa!

Organize and participate in gift exchanges online, even if you are miles apart.

Get Started

Features

User Registration

Sign up with your email or social media accounts and create a profile with your wishlist and preferences.

Secret Santa Matchmaking

The system randomly assigns participants to each other while keeping assignments secret.

Wishlist Sharing

Add gift ideas or preferences to your profile and allow for anonymous questions.

Gift Exchange

Send virtual gifts like e-cards or online gift vouchers and mark the gift as sent or received.

Chat Feature

Anonymous chat between the gift sender and recipient for hints or questions.

Countdown Timer

Show a countdown to the gift exchange date.

Group Management

Create and join private groups for Secret Santa with customizable rules, budget, and deadlines.

Social Sharing

Share your Secret Santa participation on social media (optional).

Responsive Design

Optimized for both desktop and mobile users.

How It Works

User Signup

Users sign up and create a profile with details like their wishlist, budget range, and availability.

Join/Create Groups

Users can create or join groups (e.g., "Office Secret Santa 2024"). Set group rules, budget, and deadlines.

Matchmaking

Once everyone has joined, the admin triggers the "matchmaking" process. The system assigns a Secret Santa to each user, keeping it a secret.

Gift Sharing

Users can send e-gifts or notify when physical gifts are sent. Track gift status (e.g., sent, received).

Event Day

Virtual gift-opening experience, such as a shared Zoom link or a virtual event page.
